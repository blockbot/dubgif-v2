This is a variant on the original DUBGIF site from blake x derek. The original site is an awesome concept, and I thought it would be cool to rebuild it in a way that includes loading the gifs faster, hash based url updating and on load filtering for sharing specific gif/dub combinations, to not use any third party libraries except jQuery, no flash elements, and use the HTML audio tag for the music.

This iteration was successful in meeting the goals but is limited to working in webkit browsers (Chrome, Safari, maybe the newest Opera) because of audio file type support, which can be remedied but I don't want to convert all the audio :) If you want the code, have at it here.
